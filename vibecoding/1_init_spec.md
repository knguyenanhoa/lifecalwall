# Overview
This project aims to build a Mac-compatible wallpaper which displays a life calender i.e. a rectangular grid with each cell in the grid symbolizing 1 weeks worth of time, and the entire grid symbolizes the total number of weeks in a person's life. This is the centrepiece, other elements may be added on later.

# Behavioural requirements
- Each cell in the grid symbolizes 1 week.
- The grid should be a proper rectangle, no cut corners or indents.
- The number of cells should total to the number of weeks in a typical person's lifespan of about 90 years.
- The user should be able to input/edit their birthday at any time, and the wallpaper shades in the number of weeks that have elapsed based on the current time. This should be calculated live i.e. every hour or so, and the calender is updated live.
- 30 year periods should be clearly identifiable (possibly using different colours, with shading using a different effect).

# Technical requirements
- If possible, write it in a way that is compatible with both Mac and Linux. If not possible, prioritise Mac.
